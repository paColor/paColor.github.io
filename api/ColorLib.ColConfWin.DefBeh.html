Enum ColConfWin.DefBeh

posibilités de valeur pour le flag defBeh qui indique comment doit se comporter la mise en couleur des phonèmes qui n'ont pas d'instructions de fomratage.

Namespace: ColorLib
Assembly: ColorLib.dll
Syntax
[Serializable]
public enum DefBeh

Fields

Name Description
noir

Les phonèmes pour lesquels on n'a pas de CharFormatting attribué et activé, sont mis en noir. (Les autres attributs de formatage comme 'bold' ou 'italic' ne sont pas touchés).
transparent

Les phonèmes pour lesquels on n'a pas de CharFormatting attribué et activé, ne sont pas touchés. Il restent tels qu'ils sont dans le texte d'origine.
undefined

Valeur avant l'initialisation. Comportement indéfini. Pourrait déclencher une exception.
