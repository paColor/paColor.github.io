Remarks

La partie statique de la classe gère l'attribution d'une Config à chaque fenêtre de l'interface utilisateur. L'idée est que chaque fenêtre dispose de sa propre Config que l'utilisateur peut modifier à sa guise.

Comme il n'est pas possible de recevoir un événement quand une fenêtre est fermée (en tout cas je n'ai pas trouvé comment faire), les fenêtre sont rattachées à un document (il peut y avoir plusieurs fenêtres pour le même document). L'évènement de fermeture de document peut appeler DocClosed(Object) qui libérera les ressources nécessaires.

La méthode de base de la partie statique est GetConfigFor(Object, Object) qui retourne une Config pour la fenêtre donnée. Un certain nombre d'autres méthodes permettent d'obtenir une Config par défaut ou de charger une config sauvegardée.
