UpdateLetter(Int32, Char, CharFormatting)

Met à jour la configuration pour le bouton buttonNr. Si le caractère c est déjà utilisé pour un autre bouton, la modification est refusée et la méthode retourne false. Si c est le caractère inactif ' ', le bouton est "effacé".
