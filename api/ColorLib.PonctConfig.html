Class PonctConfig

Contient la configuration pour chacune des familles de symboles de ponctuation définies dans PonctInT.

On considère qu'il existe un bouton pour chacune des familles et un bouton général qui permet de les définir tous. Ce dernier peut être dans les états master ou off.
