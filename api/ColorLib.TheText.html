Class TheText

The class TheText represents the text that is analysed and whose formatting is adapted by the analysis algorithm.

The idea is that the inheriting class is created with a string. Once this is done, different methods make it possible to apply some kind of formatting to the text. This formatting is applied to the inheriting class by calling the method SetChars see SetChars(FormattedTextEl, Config).

Remarks

Init must be called before any usage of the class. This is especially important for testing, where this could easily be forgotten.

The class is not supposed to be intiantiated directly. However, for test reasons, we do not declare it as abstract. Some members and methods of this class are public to simplify testing, although from a logical point of view, this would not be necessary.

The class is constructed in such a way that several formatting operations can be applied to it sequentially. It optimizes the computation, so that lengthy operations are only conducted once, even if two commands are excuted. Obviously this possibility will not be used often, but there are some cases where it can be useful.

SetChars(FormattedTextEl, Config)

This is to be considered as an abstract method that must be implemented by the inheriting class. The method is called for each FormattedTextEl that results from the operation applied to TheText.

The intent is that the inheriting class can then apply the requested formatting to the text in the user interface.

Is not abstract in order to make it possible to instantiate TheText and hence simplify testing.
