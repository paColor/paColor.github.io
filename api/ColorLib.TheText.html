Class TheText

The class TheText represents the text that is analysed and whose formatting is adapted by the analysis algorithm.

It is not supposed to be intiantiated directly. However, for test reasons, we do not declare it as abstract.

The idea is that the inheriting class is created with a string and a Config (see the constructor TheText(String, Config)). Once this is done, different methods make it possible to apply some kind of formatting to the text. This formatting is applied to the inheriting class by calling the method SetChars see SetChars(FormattedTextEl).

SetChars(FormattedTextEl)

This is to be considered as an abstract method that must be implemented by the inheriting class. The method is called for each FormattedTextEl that results from the operation applied to TheText.

The intend is that the inheriting class can then appy the requested formatting to the text in the user interface.

Is not abstract in order to make it possible to instantiate TheText and hence simplify testing.
