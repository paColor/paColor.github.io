CFForceBlack

CharFormatting contient un certain nombre de méthodes qui permettent de savoir comment se comporter avec une valeur false pour les différents membres de type bool. Faut-il ne rien faire ou forcer la caractéristique "gras" à "non gras" par exemple?

Une instance de CFForceBlack se comporte comme un CharFormatting mais retourne toutjours true pour les méthodes évoquées ci-dessus. Cela permet par exemple de créer un CharFormatting qui forcera la mise en noir et l'élimination des caractéristique bold, italic, underline et autres.

TheText

The class TheText represents the text that is analysed and whose formatting is adapted by the analysis algorithm.

The idea is that the inheriting class is created with a string. Once this is done, different methods make it possible to apply some kind of formatting to the text. This formatting is applied to the inheriting class by calling the method SetChars see SetChars(FormattedTextEl, Config).
